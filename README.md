# Aerion Calculator

Standalone, brand-compliant ROI/scenario calculator for Aerion. Illustrative scenarios only — not financial advice or a guarantee of returns.

## Develop
```bash
npm install
npm run dev
```

## Verify

```bash
npm run lint
npm test
npm run build
npm run qa:content
```
